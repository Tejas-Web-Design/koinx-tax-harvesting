# 🚀 KoinX Tax Loss Harvesting App 💰📊

A modern and responsive **React + TypeScript** web application that simulates a **Crypto Tax Loss Harvesting Dashboard** for investors and traders.

Built as part of the **KoinX Frontend Intern Assignment** 🧾✨

📊 Functionalities Implemented
💰 Capital Gains Summary

Displays:

Short-Term Capital Gains (STCG)
Long-Term Capital Gains (LTCG)
Net Capital Gains
Realised Capital Gains

📈 Holdings Table

Displays:

Coin Logo & Asset Name
Holdings
Average Buy Price
Current Market Price
STCG/LTCG Gains
Amount To Sell

🔄 Real-Time Harvesting Logic
Select holdings using checkboxes ☑️
Auto recalculates harvesting values
Updates profits/losses dynamically
Shows estimated tax savings 💸

👀 View All Holdings Feature

Initially limited rows are displayed.

Users can:

✅ View All Holdings
✅ Show Less Holdings

⏳ Loading State

Animated loader shown while fetching data.

📱 Responsive Design

Fully optimized for:

✅ Mobile Devices
✅ Tablets
✅ Laptops
✅ Large Screens
